# Подслушано 25 школа (бот)

Телеграм-бот для приёма сообщений в личке: копия с полными данными у `ADMIN_USER_ID`, дубликат контента в целевой чат `TARGET_GROUP_CHAT_ID`. Ответы отправителю — ответом (reply) на сообщение бота в личке админа или в целевом чате.

## Настройка

1. Создайте бота в @BotFather, вставьте токен в `.env`.
2. Узнайте свой числовой `user_id` (например @userinfobot) → `ADMIN_USER_ID`.
3. Добавьте бота в нужную группу/супергруппу, дайте право отправлять сообщения. ID чата (отрицательный для групп) → `TARGET_GROUP_CHAT_ID`.
4. `pip install -r requirements.txt` и `python bot.py`.

Пользовательские тексты бота не описывают внутреннюю маршрутизацию; правовые документы школы — вне репозитория.

**Админ:** обычные сообщения в личку боту (не reply на карточку заявки) игнорируются — чтобы случайно не записать их как «контент». Ответ отправителю — только **reply** на сообщение бота с копией/подписью заявки.
